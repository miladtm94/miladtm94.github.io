## Overview

This project addresses the physical-layer security of an Integrated Sensing and Communications (ISAC) system against a mobile proactive adversarial UAV. The base station simultaneously communicates with a legitimate user and senses/tracks the adversary, while the adversary actively maneuvers to maximize information interception.

**Key contributions:**
- Novel Stackelberg game formulation where the adversary acts as leader and the ISAC base station as follower
- Joint optimization of beamforming, artificial noise, and UAV trajectory
- Deep Reinforcement Learning (DRL) solution handling the non-cooperative, partially observable game in real time
- Significant secrecy rate and sensing accuracy gains over benchmark schemes

**Technologies:** MATLAB · Deep Reinforcement Learning · Convex Optimization · ISAC · UAV Systems
